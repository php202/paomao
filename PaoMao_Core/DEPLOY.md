# PaoMao_Core Web App 部署說明

## 外部人士／外部系統要能呼叫 Core API

**「誰可以存取」必須設為「任何人」**，否則：

- 門市預約表單 PAOPAO、員工打卡 Line@ 等專案用 UrlFetchApp 打 Core API 時會失敗（收到登入頁或 403）
- LINE Webhook 或外部點擊連結時也無法觸發

### 設定步驟

1. 開啟 **PaoMao_Core** 專案（Apps Script 編輯器）
2. 右上角 **部署** → **管理部署**
3. 找到類型為 **網路應用程式** 的那一筆，點右側 **編輯**（鉛筆圖示）
4. **誰可以存取** 改為 **「任何人」**
5. 按 **部署** 儲存

安全靠請求參數 `key=PAO_CAT_SECRET_KEY` 驗證，不依賴 Google 登入。

### 建議設定

| 項目           | 建議值     |
|----------------|------------|
| 執行身分       | 我         |
| 誰可以存取     | **任何人** |

更新程式時請用「更新既有部署」（選新版本），不要新增部署，以免網址變動。
