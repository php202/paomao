# SayDou會員全貌 預期結構

試算表：與客人問卷同試算表  
工作表名稱：`SayDou會員全貌`

## 表頭（HEADERS）

| 欄位 | 說明 |
|------|------|
| 更新時間 | 最後拉取時間 |
| 手機 | 客人手機 |
| membid | SayDou 會員 ID |
| 會員姓名 | 會員姓名 |
| 消費筆數 | 消費紀錄筆數 |
| 儲值金筆數 | 儲值／儲值金消費筆數 |
| 個人資料摘要 | 會員 view 摘要 |
| 消費紀錄摘要 | 消費全筆摘要 |
| 儲值紀錄摘要 | 儲值＋儲值金消費摘要 |
| 備註(Update) | 員工可手動填寫 |

## 本機驗證

在專案根目錄執行：

```bash
node gas/validate.js
```

會檢查 `SayDouFullProfile.js` 內 `HEADERS` 是否與上表一致。
